## Algorithm: Difficulty-Adjusted Lesson Sequencing

### Inputs

- `quiz_score`: 0–100 (average of last 3 quizzes in this subject)
- `time_on_task`: seconds spent on lesson vs. expected duration
- `error_rate`: errors / total interactions in last session
- `streak`: consecutive days active
- `cognitive_load_indicator`: from RACA epistemic layer (0.0–1.0)

### Difficulty Score Formula

```
performance_score = (quiz_score / 100) * 0.5
                  + (1 - clamp(error_rate, 0, 1)) * 0.3
                  + (1 - clamp(time_ratio - 1, 0, 1)) * 0.2

where time_ratio = time_on_task / expected_duration
```

### Difficulty Routing

| performance_score | Action                                              |
| ----------------- | --------------------------------------------------- |
| < 0.4             | Step back: recommend easier lesson in same concept  |
| 0.4 – 0.7         | Stay: recommend next lesson at same difficulty      |
| > 0.7             | Advance: recommend next lesson at higher difficulty |

### Lesson Difficulty Levels

1 (Beginner) → 2 (Emerging) → 3 (Developing) → 4 (Proficient) → 5 (Advanced)

### Cognitive Load Adjustment

If `cognitive_load_indicator > 0.7` (from RACA):

- Clamp max recommendation difficulty to current_difficulty (no advancement)
- Shorten recommended session to 15 minutes
- Flag for human educator review

### Cold Start

New learners with no history default to difficulty 2 (Emerging) and 20-minute sessions.

### Update Frequency

Recalculate after every lesson completion event.

```typescript
import { useState, useEffect, useCallback } from 'react';
import { supabase } from '../lib/supabase'; // match existing import path
import type { Lesson } from '../types/lesson';

interface AdaptiveState {
  recommendedLesson: Lesson | null;
  recommendedDifficulty: number;
  performanceScore: number | null;
  loading: boolean;
  error: string | null;
}

export function useAdaptiveLearning(userId: string, courseId?: string): AdaptiveState {
  const [state, setState] = useState<AdaptiveState>({
    recommendedLesson: null,
    recommendedDifficulty: 2,
    performanceScore: null,
    loading: true,
    error: null,
  });

  const compute = useCallback(async () => {
    if (!userId) return;
    setState(s => ({ ...s, loading: true }));

    // Fetch last 3 quiz scores from lesson_progress
    const { data: progress } = await supabase
      .from('lesson_progress')
      .select('score, time_spent_seconds, completed')
      .eq('user_id', userId)
      .order('completed_at', { ascending: false })
      .limit(3);

    // Fetch current adaptive state
    const { data: adaptiveState } = await supabase
      .from('adaptive_learning_state')
      .select('*')
      .eq('user_id', userId)
      .maybeSingle();

    if (!progress || progress.length === 0) {
      // Cold start: difficulty 2, no recommendation yet
      setState({ recommendedLesson: null, recommendedDifficulty: 2, performanceScore: null, loading: false, error: null });
      return;
    }

    // Compute performance score (see spec formula)
    const avgScore = progress.reduce((sum, p) => sum + (p.score ?? 70), 0) / progress.length;
    const quizComponent = (avgScore / 100) * 0.5;
    const errorRate = progress.filter(p => !p.completed).length / progress.length;
    const errorComponent = (1 - Math.min(errorRate, 1)) * 0.3;
    const timeComponent = 0.2; // simplified — real impl reads lesson.duration_minutes
    const performanceScore = quizComponent + errorComponent + timeComponent;

    // Determine recommended difficulty
    const currentDifficulty = adaptiveState?.recommended_difficulty ?? 2;
    let recommendedDifficulty = currentDifficulty;
    if (performanceScore < 0.4) recommendedDifficulty = Math.max(1, currentDifficulty - 1);
    else if (performanceScore > 0.7) recommendedDifficulty = Math.min(5, currentDifficulty + 1);

    // Apply cognitive load clamp
    const cognitiveLoad = adaptiveState?.cognitive_load_indicator ?? 0;
    if (cognitiveLoad > 0.7) recommendedDifficulty = Math.min(recommendedDifficulty, currentDifficulty);

    // Fetch next lesson at recommended difficulty
    const query = supabase
      .from('lessons')
      .select('*')
      .eq('difficulty_level', recommendedDifficulty)
      .order('order_index', { ascending: true })
      .limit(1);
    if (courseId) query.eq('course_id', courseId);
    const { data: lessons } = await query;

    // Persist updated state
    await supabase.from('adaptive_learning_state').upsert({
      user_id: userId,
      performance_score: performanceScore,
      recommended_difficulty: recommendedDifficulty,
      updated_at: new Date().toISOString(),
    });

    setState({
      recommendedLesson: lessons?.[0] ?? null,
      recommendedDifficulty,
      performanceScore,
      loading: false,
      error: null,
    });
  }, [userId, courseId]);

  useEffect(() => { compute(); }, [compute]);

  return state;
}
````
