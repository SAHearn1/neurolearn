  <action>
Replace stub with real implementation:
- Props: `{ sessionStartTime: Date | null, onDismiss: () => void, onTakeBreak: () => void }`
- Internal logic:
  - useEffect with setInterval (every 60 seconds) to check elapsed time
  - Show reminder after 25 minutes of continuous activity (Pomodoro technique)
  - Show "you've been going for 50 minutes" if still active after 50 minutes
  - Track if user already dismissed (store dismissed=true in state, re-show after 30 minutes)
- Renders as a non-blocking banner at the top of the page when active:
  - Message: "You've been studying for 25 minutes. Consider a 5-minute break!"
  - "Take a Break" button → calls onTakeBreak
  - "Keep Going" button → calls onDismiss (sets dismissed state)
  - role="alert" aria-live="polite" (not assertive — don't interrupt screen reader)
  - Dismiss button has aria-label="Dismiss reminder"
- Renders null (nothing) when:
  - sessionStartTime is null
  - User dismissed within last 30 minutes
  - Elapsed time < 25 minutes
- No CSS animations (keep it calm for ADHD users — no flashing or bouncing)

Pattern:
```tsx
const [show, setShow] = useState(false);
const [lastDismissed, setLastDismissed] = useState<number | null>(null);

useEffect(() => {
  if (!sessionStartTime) return;
  const interval = setInterval(() => {
    const elapsedMs = Date.now() - sessionStartTime.getTime();
    const dismissedRecently = lastDismissed && Date.now() - lastDismissed < 30 * 60 * 1000;
    setShow(elapsedMs > 25 * 60 * 1000 && !dismissedRecently);
  }, 60_000);
  return () => clearInterval(interval);
}, [sessionStartTime, lastDismissed]);

if (!show) return null;
```
